# Resume Analyzer AI

A GUI-based AI resume analyzer built using Python and Tkinter that evaluates resumes using skill matching, section detection, and ATS-style scoring.

## Features

* PDF and TXT resume support
* Skill detection with keyword matching
* Resume scoring system
* Section detection (Education, Skills, Projects, Experience)
* Suggestions for improvement

## Technologies Used

* Python
* Tkinter (GUI)
* PyPDF2 (PDF processing)

## How to Run

1. Install dependencies:
   pip install PyPDF2

2. Run the program:
   python resume_analyzer_gui.py

3. Upload your resume (.pdf or .txt) and analyze

## Output

* Resume Score
* Skills Found / Missing
* Section Analysis
* Improvement Suggestions

## Project Purpose

This project simulates an ATS (Applicant Tracking System) to evaluate resumes and provide actionable feedback for improvement.
